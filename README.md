Este proyecto de Clasificación de Residuos utiliza un modelo de inteligencia artificial entrenado por nosotros para reconocer y clasificar los tipos de residuos más comunes dentro de la universidad.
El proyecto fue realizado en Python, se utilizaron distintas librerías de python como:
- PyQt6 para la implementación de la interfaz gráfica.
- TensorFlow para poder utilizar el modelo de detección y clasificación de residuos.
- CV2 para poder acceder y trabajar con la cámara de la computadora.
